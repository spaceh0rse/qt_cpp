qfractal
========

Simple demo qt app with background worker thread

Build
-----

    qmake qfractal.pro
    make
    ./qfractal

**OR**

Open `qfractal.pro` in Qt Creator, configure, run.


========

Qt CPP Projekt BA Leipzig 2016 Alexander Heiman, Julius Krenkel
